# PFLB TEAM 3

## О проекте
Проект разработан в рамках дипломного задания по автоматизации тестирования.

Цель проекта — реализовать набор автотестов для учебного полигона **PFLB Test-API** и продемонстрировать полный цикл автоматизации: запуск тестов, проверку результатов, формирование отчетов и интеграцию

## Стек
| Компонент | Технология | Описание |
|-----------|------------|----------|
| Язык | Java | Основной язык проекта |
| Сборка | Maven | Управление зависимостями и запуском тестов |
| Тестирование | JUnit5 | Базовый тестовый фреймворк |
| UI | Selenide | UI-автотесты |
| API | RestAssured | API-автотесты |
| БД | PostgreSQL, JDBC | Проверка данных в базе |
| Логи | Log4j2 | Логирование выполнения |
| Отчеты | Allure | Генерация отчетов |
| CI/CD | GitHub Actions | Автозапуск в pipeline |
